# Quick Checklist to Find Hostinger PostgreSQL Details

## Where to Look:

1. **Hostinger Control Panel (hpanel.hostinger.com)**
   - Left sidebar → "Databases" or "Advanced" → "Databases"
   - Look for "PostgreSQL Databases" section
   - Click on your database name

2. **What to Look For:**
   - Host/Hostname (this is your DB_HOST)
   - Port (usually 5432)
   - Database Name
   - Username
   - Password

3. **If You See Only MySQL:**
   - PostgreSQL might not be available on your plan
   - Check your Hostinger plan features
   - Contact Hostinger support to enable PostgreSQL

4. **Alternative: Check via FTP/File Manager:**
   - Sometimes connection details are in config files
   - Look for files like `.env`, `wp-config.php`, or database config files

## Common Locations in Hostinger:
- Databases → PostgreSQL Databases
- Advanced → PostgreSQL
- Database Manager → PostgreSQL

## Still Stuck?
Contact Hostinger support and ask:
"I need the PostgreSQL database connection details for my account. Please provide:
- Hostname
- Port
- Database name
- Username
- Password"
